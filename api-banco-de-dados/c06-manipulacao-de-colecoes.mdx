title: '06. Manipulação de coleções'
---

Quando carregamos um objeto (seja por instanciar a classe ou carregar estaticamente), temos como retorno o objeto da classe desejada. Agora, quando carregamos coleções, temos um vetor de objetos. Da mesma maneira que podemos manipular os objetos individualmente, podemos fazer diversas operações com coleções.

Operações possíveis de serem executadas em coleções:

1.	Contar;
2.	Carregar;
3.	Carregar de forma paginada;
4.	Atualizar;
5.	Excluir.

## 6.1. O que são coleções?

Para ficar mais claro, veja o exemplo a seguir:

```php
    TTransaction::open('exemplos');

    $funcionarios = Funcionario::all();

    var_dump($funcionarios);

    TTransaction::close();
```

O comando SQL gerado é:

```sql
    SELECT nome,sobrenome,dt_nascimento,funcao_id,cidade_id,endereco,cep,id FROM funcionario
```
