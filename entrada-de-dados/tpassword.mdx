---
title: 'TPassword'
description: 'O `TPassword` é um componente de entrada de dados (derivado da classe `TField`), que permite ao usuário inserir senhas de forma segura.'
---

#### __construct($name)
Construtor da classe TPassword.

- Parâmetros:
  - `$name`: Nome do campo que será utilizado no formulário.

Exemplo de código:
```php
$senha = new TPassword('senha');
```

#### setMaxLength($length)
Define o comprimento máximo do campo.

- Parâmetros:
  - `$length`: Comprimento máximo do campo.

Exemplo de código:
```php
$senha = new TPassword('senha');
$senha->setMaxLength(16);
```

<div style="width: 180px">
{$code_tpassword_setMaxLength}
</div>

<br/>



#### setExitAction(TAction $action)
Define a ação a ser executada quando o usuário sair do campo.

- Parâmetros:
  - `$action`: Objeto TAction.

Exemplo de código:
```php
$senha = new TPassword('senha');
$senha->setExitAction(new TAction(['NomeClasseDeControle', 'onExitCampo']));
```
Código de exemplo da função a ser chamada, lembrando que essa função precisa ser estática (static function).

```php
public static function onExitCampo($param = null)
{
    new TMessage('info', print_r($param, true));
}
```

#### setExitFunction($function)
Define uma função JavaScript a ser executada quando o usuário sair do campo.

- Parâmetros:
  - `$function`: Função JavaScript.

Exemplo de código:
```php
$senha = new TPassword('senha');
$senha->setExitFunction('alert("Você saiu do campo Senha.");');
```

#### enableToggleVisibility($toggleVisibility = TRUE)
Ativa ou desativa a opção de alternar a visibilidade do campo.

- Parâmetros:
  - `$toggleVisibility` (opcional): Booleano indicando se a opção deve ser ativada (padrão: `TRUE`).

Exemplo de código:
```php
$senha = new TPassword('senha');
$senha->enableToggleVisibility();
```

{$code_tpassword_enableToggleVisibility}

#### disableAutoComplete()
Desativa o preenchimento automático do navegador.

Exemplo de código:
```php
$senha = new TPassword('senha');
$senha->disableAutoComplete();
```

#### setInnerIcon(TImage $image, $side = 'right')
Define um ícone interno ao campo.

- Parâmetros:
  - `$image`: Objeto TImage.
  - `$side` (opcional): Lado em que o ícone aparecerá (padrão: `right`).

Exemplo de código:
```php
$senha = new TPassword('senha');
$icone = new TImage('far:circle #000000');
$senha->setInnerIcon($icone);
```
