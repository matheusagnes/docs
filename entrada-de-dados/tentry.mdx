---
title: 'TEntry'
description: 'O `TEntry` é um componente de entrada de dados simples (derivado da classe `TField`), ou seja, serve para o usuário informar algum texto, como por exemplo, o nome de um produto.'

## __construct($name)
Construtor da classe TEntry.

- Parâmetros:
  - `$name`: Nome do campo que será utilizado no formulário.

Exemplo de código:
```php
$cnpj = new TEntry('cnpj');
```

#### setValue($value)
Define o valor do campo.

- Parâmetros:

Exemplo de código:
```php
$cnpj = new TEntry('cnpj');
$cnpj->setValue('00.000.000/0000-00');
```

#### setMask($mask, $replaceOnPost = FALSE)
Define a máscara do campo.

- Parâmetros:
  - `$mask`: Máscara para os dados de entrada.
  - `$replaceOnPost` (opcional): Booleano indicando se a máscara deve ser substituída após o envio do formulário (padrão: `FALSE`).

Exemplo de código:
```php
$cpf = new TEntry('cpf');
$cpf->setMask('000.000.000-00');
```

#### setNumericMask($decimals, $decimalsSeparator, $thousandSeparator, $replaceOnPost = FALSE, $reverse = FALSE, $allowNegative = TRUE)
Define a máscara numérica do campo.

- Parâmetros:
  - `$decimals`: Número de casas decimais.
  - `$decimalsSeparator`: Separador para o ponto decimal.
  - `$thousandSeparator`: Separador de milhares.
  - `$replaceOnPost` (opcional): Booleano indicando se a máscara deve ser substituída após o envio do formulário (padrão: `FALSE`).
  - `$reverse` (opcional): Booleano indicando se a máscara deve ser invertida (padrão: `FALSE`).
  - `$allowNegative` (opcional): Booleano indicando se valores negativos são permitidos (padrão: `TRUE`).

Exemplo de código:
```php
$preco = new TEntry('preco');
$preco->setNumericMask(2, ',', '.');
```

#### setExitAction(TAction $action)
Define a ação a ser executada quando o usuário sair do campo.

- Parâmetros:
  - `$action`: Objeto TAction.
  
Exemplo de código:
```php
$cep = new TEntry('cep');
$cep->setExitAction(new TAction(['NomeClasseDeControle', 'onExitCampo']))
```
Código de exemplo da função a ser chamada, lembrando que essa função precisa ser estática (static function).

```php
public static function onExitCampo($param = null)
{
    new TMessage('info', print_r($param, true));
}
```

#### reloadCompletion($field, $list, $options = null)
(Método estático) Recarrega as opções de conclusão de um campo de entrada de forma dinâmica.

- Parâmetros:
  - `$field`: Nome ou ID do campo.
  - `$list`: Lista de opções para a conclusão.
  - `$options` (opcional): Opções adicionais para a conclusão.

Exemplo de código:
```php
$genero = new TEntry('generoReloadCompletion');
$genero->setCompletion(['Feminino', 'Masculino']);

$btnGenero = new TButton('btnReloadCompletion');
$btnGenero->addStyleClass('btn-default');
$btnGenero->setImage('far:circle #000000');
$btnGenero->setAction(new TAction([$this, 'alterarOpcoes']), 'Alterar Opções');
```
Código de exemplo da função a ser chamada, lembrando que essa função precisa ser estática (static function).

```php
public static function alterarOpcoes($param = null)
{
    TEntry::reloadCompletion('generoReloadCompletion', ['Masculino', 'Feminino', 'Outro', 'Prefiro não dizer']);
}
```

#### changeMask($formName, $name, $mask)
(Método estático) Altera a máscara de um campo de entrada dinamicamente.

- Parâmetros:
  - `$formName`: Nome do formulário.
  - `$name`: Nome do campo no formulário.
  - `$mask`: Nova máscara para o campo.

Exemplo de código:
```php
$cnpj = new TEntry('cnpjChangeMask');
$cnpj->setValue('1234567800019900000');

$btnCnpj = new TButton('btnCnpjChangeMask');
$btnCnpj->addStyleClass('btn-default');
$btnCnpj->setImage('far:circle #000000');
$btnCnpj->setAction(new TAction([$this, 'alterarMascara']), 'Alterar Máscara');
```
Código de exemplo da função a ser chamada, lembrando que essa função precisa ser estática (static function).

```php
public static function alterarMascara($param = null)
{
    TEntry::changeMask('nomeForm', 'cnpjChangeMask', '00.000.000/0000-00');
}
```
