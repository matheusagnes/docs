---
title: 'TForm'
description: '`TForm` é uma classe wrapper para lidar com formulários (derivado da classe `TField`). Ela permite criar e gerenciar formulários de forma dinâmica e interativa dentro de aplicações web.'
---

## __construct($name)
Construtor da classe `TForm`.

**Parâmetros:**
  - `$name`: Nome do formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
```

<br/>

## setName($name)
Define o nome do formulário.

**Parâmetros:**
  - `$name`: Nome do formulário.

**Exemplo de código:**
```php
$formulario = new TForm();
$formulario->setName('formularioExemplo');
```

<br/>

## getName()
Retorna o nome do formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$nome = $formulario->getName();
```

<br/>

## setEditable($bool)
Define se o formulário será editável.

**Parâmetros:**
  - `$bool`: Booleano indicando se o formulário é editável.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->setEditable(true);
```

<br/>

## setTagName($name)
Define o nome da tag do formulário.

**Parâmetros:**
  - `$name`: Nome da tag.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->setTagName('div');
```

<br/>

## addField(AdiantiWidgetInterface $field)
Adiciona um campo ao formulário.

**Parâmetros:**
  - `$field`: Objeto do campo a ser adicionado.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$campo = new TEntry('nomeCampo');
$formulario->addField($campo);
```

<br/>

## delField(AdiantiWidgetInterface $field)
Remove um campo do formulário.

**Parâmetros:**
  - `$field`: Objeto do campo a ser removido.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$campo = new TEntry('nomeCampo');
$formulario->addField($campo);
$formulario->delField($campo);
```

<br/>

## delFields()
Remove todos os campos do formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->delFields();
```

<br/>

## setFields($fields)
Define os campos do formulário.

**Parâmetros:**
  - `$fields`: Array de objetos dos campos.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$campos = [new TEntry('nomeCampo1'), new TEntry('nomeCampo2')];
$formulario->setFields($campos);
```

<br/>

## getField($name)
Obtém um campo do formulário pelo nome.

**Parâmetros:**
  - `$name`: Nome do campo.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$campo = $formulario->getField('nomeCampo');
```

<br/>

## getFields()
Retorna um array com todos os campos do formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$campos = $formulario->getFields();
```

<br/>

## silentField($name)
Marca um campo como silencioso, para não ser processado automaticamente.

**Parâmetros:**
  - `$name`: Nome do campo.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->silentField('campoSilencioso');
```

## setProperty($name, $value, $replace = TRUE)
Define uma propriedade do formulário.

**Parâmetros:**
  - `$name`: Nome da propriedade.
  - `$value`: Valor da propriedade.
  - `$replace`: Substitui o valor se verdadeiro.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->setProperty('action', 'process.php');
```

<br/>

## unsetProperty($name)
Remove uma propriedade do formulário.

**Parâmetros:**
  - `$name`: Nome da propriedade.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->unsetProperty('action');
```

<br/>

## setData($object)
Define os dados do formulário a partir de um objeto.

**Parâmetros:**
  - `$object`: Objeto contendo os dados.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$dados = new StdClass();
$dados->campo = 'Valor';
$formulario->setData($dados);
```

<br/>

## getData($class = 'StdClass')
Retorna os dados do formulário como um objeto da classe especificada.

**Parâmetros:**
  - `$class`: Classe do objeto retornado.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$dados = $formulario->getData();
```

<br/>

## getValues($class = 'StdClass', $withOptions = false)
Retorna os valores iniciais do formulário como um objeto da classe especificada.

**Parâmetros:**
  - `$class`: Classe do objeto retornado.
  - `$withOptions`: Inclui opções se verdadeiro.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$valores = $formulario->getValues();
```

<br/>

## validate()
Valida os dados do formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->validate();
```

<br/>

## clear($keepDefaults = FALSE)
Limpa os dados do formulário.

**Parâmetros:**
  - `$keepDefaults`: Mantém os valores padrão se verdadeiro.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->clear();
```

<br/>

## add($object)
Adiciona um objeto ao formulário, como um container.

**Parâmetros:**
  - `$object`: Objeto a ser adicionado.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$container = new TPanel();
$formulario->add($container);
```

<br/>

## pack()
Empacota vários objetos no formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$container1 = new TPanel();
$container2 = new TPanel();
$formulario->pack($container1, $container2);
```

<br/>

## getChild()
Retorna o primeiro objeto filho do formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$child = $formulario->getChild();
```

<br/>

## getFormByName($name)
(Método estático) Retorna o objeto formulário pelo seu nome.

**Parâmetros:**
  - `$name`: Nome do formulário.

**Exemplo de código:**
```php
$formulario = TForm::getFormByName('formularioExemplo');
```

<br/>

## sendData($form_name, $object, $aggregate = FALSE, $fireEvents = TRUE, $timeout = 0)
(Método estático) Envia dados para um formulário localizado na janela pai.

**Parâmetros:**
  - `$form_name`: Nome do formulário.
  - `$object`: Objeto contendo os dados do formulário.
  - `$aggregate`: Agrega os dados se verdadeiro.
  - `$fireEvents`: Dispara eventos se verdadeiro.
  - `$timeout`: Tempo de espera antes de enviar os dados.

**Exemplo de código:**
```php
$botao = new TButton('btnEnviarDados');
$botao->setAction(new TAction(['TForm', 'sendData'], ['formularioExemplo', $objetoDados]), 'Enviar Dados');
$botao->show();
```
