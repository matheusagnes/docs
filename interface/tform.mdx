---
title: 'TForm'
description: '`TForm` é uma classe wrapper para lidar com formulários (derivado da classe `TField`). Ela permite criar e gerenciar formulários de forma dinâmica e interativa dentro de aplicações web.'
---

## __construct
Construtor da classe `TForm`.

**Parâmetros:**
<ResponseField name="$name" type="mixed" required>Nome do formulário.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
```

<br/>

## setName
Define o nome do formulário.

**Parâmetros:**
<ResponseField name="$name" type="mixed" required>Nome do formulário.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm();
$formulario->setName('formularioExemplo');
```

<br/>

## getName
Retorna o nome do formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$nome = $formulario->getName();
```

<br/>

## setEditable
Define se o formulário será editável.

**Parâmetros:**
<ResponseField name="$bool" type="mixed" required>Booleano indicando se o formulário é editável.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->setEditable(true);
```

<br/>

## setTagName
Define o nome da tag do formulário.

**Parâmetros:**
<ResponseField name="$name" type="mixed" required>Nome da tag.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->setTagName('div');
```

<br/>

## addField
Adiciona um campo ao formulário.

**Parâmetros:**
<ResponseField name="$field" type="AdiantiWidgetInterface" required>Objeto do campo a ser adicionado.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$campo = new TEntry('nomeCampo');
$formulario->addField($campo);
```

<br/>

## delField
Remove um campo do formulário.

**Parâmetros:**
<ResponseField name="$field" type="AdiantiWidgetInterface" required>Objeto do campo a ser removido.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$campo = new TEntry('nomeCampo');
$formulario->addField($campo);
$formulario->delField($campo);
```

<br/>

## delFields
Remove todos os campos do formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->delFields();
```

<br/>

## setFields
Define os campos do formulário.

**Parâmetros:**
<ResponseField name="$fields" type="mixed" required>Array de objetos dos campos.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$campos = [new TEntry('nomeCampo1'), new TEntry('nomeCampo2')];
$formulario->setFields($campos);
```

<br/>

## getField
Obtém um campo do formulário pelo nome.

**Parâmetros:**
<ResponseField name="$name" type="mixed" required>Nome do campo.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$campo = $formulario->getField('nomeCampo');
```

<br/>

## getFields
Retorna um array com todos os campos do formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$campos = $formulario->getFields();
```

<br/>

## silentField
Marca um campo como silencioso, para não ser processado automaticamente.

**Parâmetros:**
<ResponseField name="$name" type="mixed" required>Nome do campo.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->silentField('campoSilencioso');
```

## setProperty
Define uma propriedade do formulário.

**Parâmetros:**
<ResponseField name="$name" type="mixed" required>Nome da propriedade.</ResponseField>
<ResponseField name="$value" type="mixed" required>Valor da propriedade.</ResponseField>
<ResponseField name="$replace" type="bool" default="TRUE">(Opcional) Substitui o valor se verdadeiro.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->setProperty('action', 'process.php');
```

<br/>

## unsetProperty
Remove uma propriedade do formulário.

**Parâmetros:**
<ResponseField name="$name" type="mixed" required>Nome da propriedade.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->unsetProperty('action');
```

<br/>

## setData
Define os dados do formulário a partir de um objeto.

**Parâmetros:**
<ResponseField name="$object" type="mixed" required>Objeto contendo os dados.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$dados = new StdClass();
$dados->campo = 'Valor';
$formulario->setData($dados);
```

<br/>

## getData
Retorna os dados do formulário como um objeto da classe especificada.

**Parâmetros:**
<ResponseField name="$class" type="string" default="StdClass">(Opcional) Classe do objeto retornado.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$dados = $formulario->getData();
```

<br/>

## getValues
Retorna os valores iniciais do formulário como um objeto da classe especificada.

**Parâmetros:**
<ResponseField name="$class" type="string" default="StdClass">(Opcional) Classe do objeto retornado.</ResponseField>
<ResponseField name="$withOptions" type="bool" default="false">(Opcional) Inclui opções se verdadeiro.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$valores = $formulario->getValues();
```

<br/>

## validate
Valida os dados do formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->validate();
```

<br/>

## clear
Limpa os dados do formulário.

**Parâmetros:**
<ResponseField name="$keepDefaults" type="bool" default="FALSE">(Opcional) Mantém os valores padrão se verdadeiro.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$formulario->clear();
```

<br/>

## add
Adiciona um objeto ao formulário, como um container.

**Parâmetros:**
<ResponseField name="$object" type="mixed" required>Objeto a ser adicionado.</ResponseField>

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$container = new TPanel();
$formulario->add($container);
```

<br/>

## pack
Empacota vários objetos no formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$container1 = new TPanel();
$container2 = new TPanel();
$formulario->pack($container1, $container2);
```

<br/>

## getChild
Retorna o primeiro objeto filho do formulário.

**Exemplo de código:**
```php
$formulario = new TForm('formularioExemplo');
$child = $formulario->getChild();
```

<br/>

## getFormByName
(Método estático) Retorna o objeto formulário pelo seu nome.

**Parâmetros:**
<ResponseField name="$name" type="mixed" required>Nome do formulário.</ResponseField>

**Exemplo de código:**
```php
$formulario = TForm::getFormByName('formularioExemplo');
```

<br/>

## sendData
(Método estático) Envia dados para um formulário localizado na janela pai.

**Parâmetros:**
<ResponseField name="$form_name" type="mixed" required>Nome do formulário.</ResponseField>
<ResponseField name="$object" type="mixed" required>Objeto contendo os dados do formulário.</ResponseField>
<ResponseField name="$aggregate" type="bool" default="FALSE">(Opcional) Agrega os dados se verdadeiro.</ResponseField>
<ResponseField name="$fireEvents" type="bool" default="TRUE">(Opcional) Dispara os eventos se verdadeiro.</ResponseField>
<ResponseField name="$timeout" type="int" default="0">(Opcional) Tempo de espera em milissegundos (ms) antes de enviar os dados.</ResponseField>

**Exemplo de código:**
```php
$botao = new TButton('btnEnviarDados');
$botao->setAction(new TAction(['TForm', 'sendData'], ['formularioExemplo', $objetoDados]), 'Enviar Dados');
$botao->show();
```
