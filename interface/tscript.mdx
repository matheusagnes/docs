## create
(Método estático) Cria um script JavaScript para ser executado na página.

**Parâmetros:**
<ResponseField name="$code" type="mixed" required>Código-fonte JavaScript a ser inserido.</ResponseField>
<ResponseField name="$show" type="bool" default="TRUE">(Opcional) Booleano que define se o script será mostrado imediatamente.</ResponseField>
<ResponseField name="$timeout" type="mixed" default="null">(Opcional) Tempo em milissegundos após o qual o script será executado.</ResponseField>

**Exemplo de código:**
```php
TScript::create('alert("Olá, mundo!");', true, 3000);
```
