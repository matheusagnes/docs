---
title: 'TScript'
description: 'O `TScript` é uma classe base para scripts, permitindo a inserção e manipulação de scripts JavaScript em uma aplicação (derivado da classe `TElement`).'
---

## __construct
Construtor da classe `TScript`.

**Parâmetros:**
<ResponseField name="$code" type="mixed" required>Código-fonte JavaScript a ser inserido.</ResponseField>
<ResponseField name="$show" type="bool" default="TRUE">(Opcional) Booleano que define se o script será mostrado imediatamente.</ResponseField>
<ResponseField name="$timeout" type="mixed" default="null">(Opcional) Tempo em milissegundos após o qual o script será executado.</ResponseField>

**Exemplo de código:**
```php
$codigoJavaScript = new TScript('alert("Olá, mundo!");', true, 3000);
```

<br/>

## importFromFile
(Método estático) Importa um arquivo de script.

**Parâmetros:**
<ResponseField name="$script" type="mixed" required>Nome do arquivo de script a ser importado.</ResponseField>
<ResponseField name="$show" type="bool" required>(Opcional) Booleano que define se o script será mostrado imediatamente.</ResponseField>
<ResponseField name="$timeout" type="mixed" required>(Opcional) Tempo em milissegundos após o qual o script será executado.</ResponseField>

**Exemplo de código:**
```php
$btnImportarScript = new TButton('btnImportarScript');
$btnImportarScript->addStyleClass('btn-default');
$btnImportarScript->setImage('far:circle #000000');
$btnImportarScript->setAction(new TAction(['TScriptViewController', 'importarScript'], ['script.js']), 'Importar Script');
```
