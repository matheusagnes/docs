---
title: 'TScript'
description: 'O `TScript` é uma classe base para scripts, permitindo a inserção e manipulação de scripts JavaScript em uma aplicação (derivado da classe `TElement`).'
---

## __construct($code, $show = TRUE, $timeout = null)
Construtor da classe TScript.

**Parâmetros:**
  - `$code`: Código fonte JavaScript a ser inserido.
  - `$show`: Booleano que define se o script será mostrado imediatamente (padrão: `TRUE`).
  - `$timeout`: Tempo em milissegundos após o qual o script será executado (opcional).

**Exemplo de código:**
```php
$codigoJavaScript = new TScript('alert("Olá, mundo!");', true, 3000);
```

<br/>

## importFromFile($script, $show = TRUE, $timeout = null) (Método estático)
Importa um arquivo de script.

**Parâmetros:**
  - `$script`: Nome do arquivo de script a ser importado.
  - `$show`: Booleano que define se o script será mostrado imediatamente (padrão: `TRUE`).
  - `$timeout`: Tempo em milissegundos após o qual o script será executado (opcional).

**Exemplo de código:**
```php
$btnImportarScript = new TButton('btnImportarScript');
$btnImportarScript->addStyleClass('btn-default');
$btnImportarScript->setImage('far:circle #000000');
$btnImportarScript->setAction(new TAction(['TScriptViewController', 'importarScript'], ['script.js']), 'Importar Script');
```
